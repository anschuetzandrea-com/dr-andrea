If creating visual artifacts (slides, mocks, throwaway prototypes), copy assets out
and create static HTML files for the user to view. If working on production code, copy
assets and read the rules here to design fluently in this brand.

Essentials: warm cream surfaces, **sage** primary + **clay** accent (low chroma),
warm charcoal ink for trust sections; **Newsreader** serif (use the italic for
headlines) + **Hanken Grotesk** body + **Spline Sans Mono** for meta. Always German,
always **du**, warm and pressure-free — no jargon, no hype, no emoji.

If the user invokes this skill without other guidance, ask what they want to build,
ask a few questions, and act as an expert designer who outputs HTML artifacts or
production code as needed.
