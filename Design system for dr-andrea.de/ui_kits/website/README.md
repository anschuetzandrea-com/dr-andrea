# Website UI Kit — Dr. Andrea (Rebrand)

A high-fidelity recreation of the dr-andrea.de landing page, **rebranded** into the
"Calm, grounded, warm-editorial" system. Same content and structure as the live
site, fresh aesthetic.

## Files
- `index.html` — mounts the homepage. Open directly or via the Design System tab.
- `Homepage.jsx` — the full page, composing design-system components
  (`Button`, `TestimonialCard`, `StepItem`, `MediaSlot`, `Card`, `Badge`,
  `Eyebrow`, `Input`, `Textarea`).

## Sections (mapped from the original)
1. Nav + wordmark
2. Hero — "Finde dich selbst" → italic serif headline, portrait slot
3. Newsletter band (sage)
4. Testimonials — Boris & Edgar (alternating sides)
5. "Kann ich dir helfen?" split (inverse CTA + bio)
6. "Gib dir eine Chance" — numbered steps + media
7. Trust quote band (inverse ink)
8. Contact form
9. Footer

## Notes
- All imagery is `MediaSlot` placeholders — drop in the real portrait, lifestyle
  photos, and testimonial videos for production.
- Copy is German and uses the du-form, matching the practice's voice.
