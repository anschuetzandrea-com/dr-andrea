Pill-shaped call-to-action button — sage primary, clay accent, sage-outline secondary, quiet ghost. Use `primary` for the main booking CTA ("Starte jetzt mit deinem kostenlosen Erstgespräch").

```jsx
<Button variant="primary" size="lg" href="https://calendly.com/...">
  Starte jetzt mit deinem kostenlosen Erstgespräch
</Button>
```

Variants: `primary` (sage fill) · `accent` (clay fill, for emotional moments) · `secondary` (sage outline) · `ghost` (quiet, on busy backgrounds). Sizes `sm | md | lg`. Set `pill={false}` for a soft-rectangle button; `fullWidth` for forms.
