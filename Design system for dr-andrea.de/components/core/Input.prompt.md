Single-line text field for the contact form and newsletter signup. Warm white fill, hairline border, sage focus ring.

```jsx
<Input label="Name" placeholder="Dein Name" />
<Input label="Email" type="email" error="Bitte gib eine gültige Email an" />
```

Pass `label`, `hint`, or `error`. All native input attributes pass through. Pair with `Textarea` for the message field.
